# Theme System

Themes allow lightweight styling metadata.

## Properties

- `background`
- `foreground`
- `accent`
- `font`
- `spacing`
- `radius`

## Selectors and attributes

Selectors: `quiz`, `question`, `answer`, `correct`, `wrong`

Attributes:
- `font`
- `size`
- `color`
- `weight`
- `spacing`
